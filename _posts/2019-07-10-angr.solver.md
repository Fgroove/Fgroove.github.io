# angr.solver

AST:符号执行产生的抽象语法树，可以解释为SMT解析器中的约束条件。

解决问题“已知输出，输入应该满足什么条件？”

## 位向量

位向量就是一个比特序列，可以解释为数学上的有界整数。

* BVV：Bitvector value
* BVS：Bitvector symbolic

```python
In [14]: s = start_state

In [15]: one = s.solver.BVV(1,64) # 值，长度

In [16]: one
Out[16]: <BV64 0x1> # BV64代表长度64
```
